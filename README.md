###### 2022년 09월 12일  
<span class="article-text"><p>WeatherKit은 사용자가 최신 정보를 확인하고 안전하게 미리 대비하는 데 도움이 되는 여러 가지 유용한 기상 데이터를 앱과 서비스에 제공합니다.
