## App Store submissions now open for the latest OS releases  

###### September 9, 2024  
<div class="article-text"><p>iOS18, iPadOS18, macOSSequoia, tvOS18, visionOS2, and watchOS11 will soon be available to customers worldwide. Build your apps and games using the Xcode16 Release Candidate and latest SDKs, test them using TestFlight, and submit them for review to the AppStore. You can now start deploying seamlessly to TestFlight and the AppStore from XcodeCloud. With exciting new features like <a href="https://developer.apple.com/watchos/">watchOS Live Activities</a>, <a href="https://developer.apple.com/design/human-interface-guidelines/app-icons#iOS-iPadOS">app icon customization</a>, and powerful updates to Swift, Siri, Controls, and CoreML, you can deliver even more unique experiences on Appleplatforms.</p><p>And beginning next month, you’ll be able to bring the incredible new features of <a href="https://developer.apple.com/apple-intelligence/">AppleIntelligence</a> into your apps to help inspire the way users communicate, work, and express themselves.</p><p>Starting April 2025, apps uploaded to AppStoreConnect must be built with SDKs for iOS18, iPadOS18, tvOS18, visionOS2, or watchOS11.</p><p><a href="https://developer.apple.com/app-store/submitting/">Learn about <span class="icon icon-after icon-chevronright nowrap">submitting apps</span></a></p></div>
